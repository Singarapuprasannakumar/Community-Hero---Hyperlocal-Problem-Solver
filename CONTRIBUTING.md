# Contributing to Community Hero AI

We welcome developer contributions!

## Branch Guidelines
1. Fork the repo.
2. Build new features on a branch: `feature/your-feature-name`.
3. Submit a PR against the `main` branch.

## Checklist Before Submitting PR
- Verify code compiles cleanly with strict types: `npm run build`.
- Confirm unit checks pass.
- Write tests where appropriate.
