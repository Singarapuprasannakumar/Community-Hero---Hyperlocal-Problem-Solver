# Security Policy

## Supported Versions

Only the latest release (v1.0.0) is actively supported.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

Please report vulnerabilities to the security coordinate officer at `security@communityhero.ai`. We aim to respond within 48 hours.
